# تتبع الوقت 🕒

**تتبع الوقت** هو تطبيق بسيط مصمم لتتبع ساعات عملك بطريقة سهلة وتحترم الخصوصية. يتيح لك عرض نشاطك اليومي والأسبوعي والشهري دون أي تعقيدات.

🚀 **عرض مباشر**: [suivi-de-temps.lapierrequimousse.com](https://suivi-de-temps.vercel.app)

## 🌟 المزايا

- **مفتوح المصدر**: الكود شفاف ومتاح للجميع.
- **بدون حساب**: لا يتطلب التسجيل.
- **بيانات محلية**: تبقى جميع بياناتك على جهاز الكمبيوتر الخاص بك (مخزنة محلياً في متصفحك).
- **قابل للاستضافة الذاتية**: قم بتثبيته على بنيتك التحتية الخاصة بسهولة تامة.
- **متعدد اللغات**: متوفر بعدة لغات.

## 🚀 البدء السريع

### المتطلبات الأساسية

- Node.js (يوصى بالإصدار 18 أو أعلى)
- pnpm (موصى به) أو npm / yarn

### تثبيت التبعيات

```bash
pnpm install
```

### التشغيل في وضع التطوير

```bash
pnpm dev
```

سيكون التطبيق متاحاً على `http://localhost:3000`.

## 📦 الإنتاج والنشر

### البناء القياسي (Node.js)

لبناء التطبيق لبيئة Node.js:

```bash
pnpm build
```

### التوليد الاستاتيكي (SSG)

لتوليد نسخة استاتيكية من التطبيق:

```bash
pnpm generate
```

**لماذا تستخدم التوليد الاستاتيكي؟**

- **بساطة الاستضافة**: يمكن استضافة الملفات الناتجة في مجلد `.output/public` على أي مزود استضافة.
- **الأداء**: يتم تقديم الصفحات مسبقاً، مما يتيح تحميلاً فورياً تقريباً.
